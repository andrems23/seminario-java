# Seminário Java – Recursividade e Algoritmos de Ordenação

Projeto desenvolvido em **Java** com foco no estudo de **recursividade** e **algoritmos de ordenação**, abordando conceitos fundamentais da análise e implementação de algoritmos.

## Sobre o Projeto

Este repositório reúne implementações práticas em Java para estudo de:

- Recursividade
- Métodos de ordenação básicos
- Métodos de ordenação avançados
- Análise de desempenho de algoritmos

O objetivo é compreender como diferentes algoritmos funcionam e comparar sua eficiência em diferentes cenários.

## Conteúdo Implementado

### Recursividade

Implementações utilizando chamadas recursivas para resolução de problemas clássicos:

- Fatorial
- Sequência de Fibonacci
- Soma de elementos
- Busca recursiva
- Potenciação

### Métodos de Ordenação

#### Métodos Fracos
Algoritmos mais simples, geralmente menos eficientes para grandes volumes de dados:

- Bubble Sort
- Selection Sort
- Insertion Sort

#### Métodos Fortes
Algoritmos mais eficientes e utilizados em aplicações reais:

- Quick Sort
- Merge Sort
- Heap Sort
